有理数的定义

难度：简单

整数和分数统称为有理数。有理数可以用分数的形式表示，其中分子和分母都是整数，且分母不为零。

📌 典型例题
下列各数中，哪些是有理数？-3, 0, 1/2, 0.25, π
答案：-3, 0, 1/2, 0.25 是有理数
解析：根据有理数的定义，整数和分数统称为有理数。-3是负整数，0是整数，1/2是分数，0.25可以表示为1/4也是分数，它们都是有理数。而π是无限不循环小数，不能表示为分数形式，所以不是有理数。

有理数的加法法则

难度：中等

同号两数相加，取相同的符号，并把绝对值相加；异号两数相加，取绝对值较大的数的符号，并用较大的绝对值减去较小的绝对值；互为相反数的两个数相加得0；一个数同0相加，仍得这个数。

加法法则： a + b = b + a (交换律)
(a + b) + c = a + (b + c) (结合律)
📌 典型例题
计算：(-3) + (+5) + (-7) + (+9)
答案：4
解析：方法一：按顺序计算
(-3) + (+5) = +2
(+2) + (-7) = -5
(-5) + (+9) = +4

方法二：运用加法交换律和结合律，把同号的数结合在一起
[(-3) + (-7)] + [(+5) + (+9)]
= (-10) + (+14)
= +4

绝对值的化简方法

难度：中等

正数的绝对值是它本身，负数的绝对值是它的相反数，0的绝对值是0。在化简含有绝对值的表达式时，首先要判断绝对值符号内的数或式的正负性，然后根据绝对值的定义去掉绝对值符号。

绝对值定义： |a| = { a (a≥0) ; -a (a<0) }
